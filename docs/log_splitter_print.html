DPDT Relay Pin Usage

K1Pole 1 (K1-1): NO contact used as seal — latches cycle after CYCLE button released
Pole 2 (K1-2): NO contact feeds +12V to K2 coil circuit — only during auto cycle
K2Pole 1 (K2-1): Changeover (COM/NO/NC) routes power to SOL-A (energized) or SOL-B (de-energized)
Pole 2: Unused / spare

Sequence of Operation

IDLECylinder at home. LS2 closed. K1 off, K2 off. No solenoids powered.
FWDHold FWD button → +12V goes direct to SOL-A → cylinder extends. Release to stop. Bypasses all relay logic.
BWDHold BWD button → +12V goes direct to SOL-B → cylinder retracts. Release to stop. Bypasses all relay logic.
CYCLEPress CYCLE → K1 energizes → K1-1 seals (latches) → K1-2 closes, feeding K2 coil through LS1 → K2 energizes → K2-1 changeover flips to NO → SOL-A fires → cylinder extends.
SPLITCylinder reaches full extend → LS1 opens → K2 drops → K2-1 changeover falls to NC → SOL-B fires → cylinder retracts automatically.
HOMECylinder reaches full retract → LS2 opens → K1 unlatches → K1-1 opens (seal broken) → K1-2 opens → all power removed. Cycle complete.
E-STOPPress E-Stop at any time → NC contact opens → kills entire +12V bus → all circuits dead immediately. Twist to release.

Limit Switch Wiring Detail

LS1Mount where cylinder rod or wedge reaches full extension. NC contact wired in series with K2 coil. When cylinder hits LS1, contact opens, K2 drops, retract begins.
LS2Mount where cylinder rod reaches full retract (home). NC contact wired in series with K1 coil. When cylinder reaches home, contact opens, K1 drops, cycle ends.
Both switches should be NC type — fail-safe: broken wire = switch "open" = cycle stops.
WIRING NOTES:
• Use 12 AWG minimum for solenoid feeds and main +12V bus. 16 AWG acceptable for relay coil circuits and signal wiring.
• Install 1N4007 flyback diode across EVERY inductive coil (K1, K2, SOL-A, SOL-B). Cathode band toward +12V side. Prevents relay contact arcing and pitting.
• E-Stop NO contact is unused in this circuit — available for pilot light, horn, or shutdown indicator.
• FWD and BWD buttons are wired direct-to-solenoid, completely independent of relay logic. If a relay fails, manual jog still works.
• OPTIONAL INTERLOCK: Wire FWD and BWD buttons through a K1 NC contact to disable jog during auto cycle. Prevents operator from fighting the auto-return.
• OPTIONAL SAFETY TIMER: Add a 12VDC time-delay relay (15s TON) in series with K1 coil. If LS1 never trips (jammed log, broken switch), timer kills the cycle automatically.
• K2 changeover guarantees only one solenoid is ever powered during auto cycle — mechanical interlock built into the relay.
